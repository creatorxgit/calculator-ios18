# Calculator IOS 18
A simple calculator on python with IOS 18 style
How to run?
1.Press Windows + R, then write cmd and press Enter(or click OK)
2.Write this command: pip install python(if you havent got python on your computer)
2.Write this command: pip install tkinter
3.Press on The lates release right down of the official page on GitHub.
4.Go Down and click on windows.py
5.Open the file.
